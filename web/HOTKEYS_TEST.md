# Инструкция по проверке горячих клавиш

## Быстрый способ проверить все горячие клавиши

1. **Откройте справку по горячим клавишам:**
   - Нажмите `Ctrl + /` (или `Ctrl + Shift + 7` на некоторых раскладках)
   - Откроется модальное окно со списком всех доступных горячих клавиш

## Список горячих клавиш для проверки

### 1. Поиск по каталогу
- **Комбинация:** `Ctrl + K`
- **Действие:** Фокусирует поле поиска на странице каталога
- **Как проверить:**
  1. Откройте любую страницу сайта
  2. Нажмите `Ctrl + K`
  3. Если на странице есть поле поиска - оно должно получить фокус
  4. Если поля поиска нет - произойдет переход на страницу каталога

### 2. Открыть корзину
- **Комбинация:** `Ctrl + C`
- **Действие:** Переход на страницу корзины
- **Как проверить:**
  1. Нажмите `Ctrl + C`
  2. Должен произойти переход на `/cart/`

### 3. Открыть заказы
- **Комбинация:** `Ctrl + O`
- **Действие:** Переход на страницу заказов
- **Как проверить:**
  1. Нажмите `Ctrl + O`
  2. Должен произойти переход на `/orders/`

### 4. Открыть аналитику
- **Комбинация:** `Ctrl + A`
- **Действие:** Переход на страницу аналитики (для Admin/Analyst)
- **Как проверить:**
  1. Войдите как администратор или аналитик
  2. Нажмите `Ctrl + A`
  3. Должен произойти переход на `/analytics/`

### 5. Закрыть модальные окна
- **Комбинация:** `Esc`
- **Действие:** Закрывает все открытые модальные окна, dropdown меню и алерты
- **Как проверить:**
  1. Откройте любое модальное окно (например, справку по горячим клавишам)
  2. Нажмите `Esc`
  3. Модальное окно должно закрыться

### 6. Сохранить форму
- **Комбинация:** `Ctrl + S`
- **Действие:** Отправляет форму на странице (если есть)
- **Как проверить:**
  1. Откройте страницу с формой (например, настройки)
  2. Измените какие-либо значения
  3. Нажмите `Ctrl + S`
  4. Форма должна отправиться

### 7. Показать справку
- **Комбинация:** `Ctrl + /`
- **Действие:** Открывает модальное окно со списком всех горячих клавиш
- **Как проверить:**
  1. Нажмите `Ctrl + /`
  2. Должно открыться модальное окно со списком горячих клавиш

### 8. Экспорт/печать
- **Комбинация:** `Ctrl + P`
- **Действие:** Открывает диалог печати или нажимает кнопку экспорта
- **Как проверить:**
  1. Нажмите `Ctrl + P`
  2. Должен открыться диалог печати браузера

### 9. Перейти на главную
- **Комбинация:** `Ctrl + H`
- **Действие:** Переход на главную страницу
- **Как проверить:**
  1. Нажмите `Ctrl + H`
  2. Должен произойти переход на `/`

### 10. Открыть настройки
- **Комбинация:** `Ctrl + ,` (Ctrl + запятая)
- **Действие:** Переход на страницу настроек
- **Как проверить:**
  1. Нажмите `Ctrl + ,`
  2. Должен произойти переход на `/settings/`

## Проверка через консоль браузера

Вы также можете проверить, что модуль горячих клавиш загружен, открыв консоль браузера (F12) и выполнив:

```javascript
// Проверить, что модуль загружен
console.log(window.hotkeys);

// Посмотреть список всех горячих клавиш
console.log(window.hotkeys.keys);

// Вручную вызвать действие (например, показать справку)
window.hotkeys.keys.help.action();
```

## Важные замечания

1. **Горячие клавиши не работают в полях ввода:**
   - Если курсор находится в поле `<input>` или `<textarea>`, большинство горячих клавиш не сработают
   - Исключение: `Esc` и `Ctrl + /` работают всегда

2. **Конфликты с браузером:**
   - Некоторые комбинации могут конфликтовать с горячими клавишами браузера
   - Например, `Ctrl + P` обычно открывает диалог печати браузера (это нормально)

3. **Проверка на разных страницах:**
   - Некоторые горячие клавиши работают только на определенных страницах
   - Например, `Ctrl + S` работает только если на странице есть форма

## Отладка

Если горячие клавиши не работают:

1. Откройте консоль браузера (F12)
2. Проверьте, нет ли ошибок JavaScript
3. Убедитесь, что файл `hotkeys.js` загружен:
   ```javascript
   console.log(typeof window.hotkeys); // должно быть "object"
   ```
4. Проверьте, что обработчик событий зарегистрирован:
   ```javascript
   // Попробуйте вручную вызвать обработчик
   window.hotkeys.handleKeyPress(new KeyboardEvent('keydown', {key: 'k', ctrlKey: true}));
   ```
